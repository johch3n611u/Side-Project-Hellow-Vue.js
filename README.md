# Try some Vue.js

## 序

以學過 Angular 的角度來看 Vue 上手速度算是蠻快的，基本上熟悉文件配置，使用方式與組成都很類似，

差別在於 ng 每次都需要 Ahead-of-Time 或 Just-in-Time 將 TS 編譯為瀏覽器能讀懂的程式碼，而 Vue 則是像函式庫一樣的引用

在這簡易做一個 todolist demo ，有時間在上到能夠在 firebase 儲存狀態。
